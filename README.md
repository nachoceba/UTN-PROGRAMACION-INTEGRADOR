DESCRIPCION DEL PROGRAMA:

Desarrollamos una aplicación en Python para gestionar información de distintos países utilizando archivos CSV, listas, diccionarios y funciones. El programa carga automáticamente los datos desde paises.csv y los almacena en una lista de diccionarios, permitiendo trabajar con ellos de forma ordenada.

A partir de esa información, implementamos un menú interactivo que le permite al usuario realizar varias acciones: buscar un país, filtrar resultados según características, ordenar la lista por diferentes criterios, mostrar estadísticas generales y agregar nuevos países al archivo. Cada opción del menú activa una función independiente, lo que nos permitió modularizar el código y mantener una estructura clara.

Además, agregamos validaciones para asegurarnos de que el usuario ingrese opciones correctas y evitar errores durante la ejecución.

INSTRUCCIONES GENERALES:

El programa se maneja a través de un menú principal donde el usuario puede buscar países, filtrarlos, ordenarlos, ver estadísticas o añadir nuevos registros. Después de realizar cualquier acción, el sistema vuelve al menú para seguir trabajando. En caso de ingresar una opción inválida, el programa muestra un mensaje de error y solicita una nueva entrada. Para salir, simplemente se elige la opción correspondiente.

1) Ejecutar el programa.

2) Elegir una opción del menú (A–F).

3) Segun la opcion elegida proporcionar los datos que piden por pantalla.

4) Una vez terminada la tarea elegida indicar al sistema si desea seguir en la misma o volver al menu.

5) Seleccionar F para salir del sistema.
